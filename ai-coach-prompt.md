# 🤖 AI 词汇活化教练 — Prompt 合集

以下 prompt 配合 `flashcards.html` 使用。闪卡自测后，用 AI 帮你**纠错、扩展、巩固**。

---

## 1. 每日活化练习（核心）

```
你是一个英语词汇活化教练。我的词汇量约 3500，属于 B1 水平，
核心问题是"认得但不会主动用"。

请你按以下流程帮我练习：

第一步：给我一个 B1 级别的高频词和它的英文释义（用简单英文解释，不超过 15 个词）。
第二步：给我一句生活中常用的例句。
第三步：让我用这个词造一个自己的句子。
第四步：【等我回复后】纠正我的句子：
  - 如果有语法错误，标注出来并解释
  - 如果表达不自然（像中文直译），给出 native speaker 会怎么说
  - 如果完全正确，告诉我并且给一个"升级版"——用这个词的更高级用法再造一个句子

每轮 1 个词，总共练 10 个词。开始吧。
```

---

## 2. 闪卡后的"深度巩固"

把闪卡中标记为"还需练习"的词列出来，粘贴到这个 prompt 里：

```
以下是 5 个我还不熟练的单词：

[粘贴单词列表]

请帮我为每个词做三件事：
1. 用 B1 水平的简单英文解释（不是中文翻译）
2. 造 3 个不同生活场景的例句（工作、社交、日常生活各一个）
3. 指出一个中国学习者常见的搭配错误，并纠正

全部用英文回复，我遇到不懂的地方会单独问。
```

---

## 3. 口语输出——用新词聊天（ChatGPT 语音模式）

```
让我们用我刚学的词进行一次对话。

我刚练了这些词：[粘贴今天学的 10 个词]

请你扮演一个朋友，我们随便聊聊今天的工作/生活。
对话中你自然地使用我刚学的这些词中的 3-5 个，
如果我用错了某个词，在对话结束后告诉我。

用日常对话的语气，不要说教。开始吧。
```

---

## 4. 主题造句挑战

```
给我一个贴近日常生活的主题（比如：上班通勤、点外卖、和朋友吵架、旅行计划）。

我会围绕这个主题造 5 个句子，每个句子必须使用你指定的词。
你先列出 5 个词，然后我逐个造句，你逐个纠正。

开始。
```

---

## 5. "中文→英文"反应速度训练

适合在手机上快速刷，训练大脑不翻译直接输出：

```
请快速测试我的反应速度。

规则：
1. 你说一个中文词/短句（B1 难度日常用语）
2. 我在 3 秒内说出英文
3. 如果我说对了，立刻下一个
4. 如果我卡住了或说错了，告诉我最自然的英文说法

说 20 个就停，然后告诉我：
- 正确率
- 慢在哪类表达上（动词搭配？介词？时态？）
- 给我 2 个针对性练习建议

只做 20 个，保持快速节奏。开始！
```

---

## 6. 写作输出——日报式（每周 3 次）

```
请做我的英语写作教练。我会用英文写 100-200 字的"今日小结"。

批改规则：
🔴 语法错误 → 标注并解释原因
🟡 不地道的表达 → 给出 native speaker 版本
🟢 写得不错的 → 表扬，鼓励我多用类似表达

改完后问 2 个问题引导我继续写更多。

开始，我写好了：[粘贴你的英文日记]
```

## 🗓 推荐使用节奏

| 时段 | 做什么 | 用哪个工具 |
|------|--------|-----------|
| 早晨 10min | 闪卡自测 10 个词 | `flashcards.html` |
| 午休 10min | 把薄弱词丢给 AI 深度巩固 | Prompt #2 |
| 晚上 10min | 语音模式口语对话 | Prompt #3 + ChatGPT 语音 |

**每天 30 分钟，坚持 21 天先。**
